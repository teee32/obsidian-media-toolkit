# Issue 3: README 可信度问题 - Issues 链接是占位符

## 严重程度: P0 - 影响可信度

### 问题描述
README 的反馈链接写的是 `https://github.com/your-repo/obsidian-media-manager/issues` 这种模板占位符，而非真实地址。

### 影响
这类细节会让人立刻判断：作者没有把"对外可用"当成目标，更像是练手/临时放上来。

### 修复建议
- 将 README 中的 issues 链接改为真实地址: `https://github.com/teee32/obsidian-media-manager/issues`
- 安装方式写清楚（如 BRAT / manual），别写"推荐市场安装"这种当前证据不支持的内容
